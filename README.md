# Ethernal

> AI-native prediction markets platform built on Rain Protocol (Arbitrum One)

## 🌟 Overview

Ethernal is a cutting-edge prediction markets platform that leverages AI-powered market generation and social features. Built on top of the Rain Protocol, it aims to revolutionize the way people engage with prediction markets.

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Blockchain**: Arbitrum One, Rain Protocol
- **Smart Contracts**: Solidity
- **Development**: wagmi, viem, RainbowKit, Alchemy AA
- **Database**: Prisma
- **State Management**: Zustand
- **Charts**: Recharts

## 🚀 Features

- **AI Market Generator**: AI-powered market creation
- **Social Feed**: Community-driven content
- **Creator Economy**: Tools for market creators
- **Leaderboards**: Performance tracking
- **LP Dashboard**: Liquidity provider tools
- **Tournament**: Competitive prediction challenges
- **Whale Analysis**: Market insights and analytics
- **Gasless Transactions**: Seamless user experience

## 📊 Project Status

- **Phase**: Early development
- **Target**: Rain Builders Program grant
- **Initial Funding**: $500
- **Strategy**: Focus on short-term markets for maximum volume

## 🎯 Goals

- Build a user-friendly prediction markets interface
- Create innovative AI-powered market generation
- Establish a strong community presence
- Develop sustainable revenue models through transaction fees
